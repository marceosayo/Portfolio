## Installation

Follow these steps:

1. Make sure you have the following installed on your machine:

```
- Git
- NodeJS
- npm (Node Package Manager)
```

2. Cloning the Repository

```
git clone https://github.com/marceosayo/Portfolio.git
cd website-portfolio
npm install
npm run dev
```
